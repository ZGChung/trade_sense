# TradeSense 📈

**交易直觉训练游戏** - 用历史事件预测股票走势，训练你的市场直觉！

## 🎮 游戏定位

TradeSense 是一款**娱乐导向的金融模拟游戏**，不是教学工具：
- 通过历史真实事件学习市场规律
- 挑战排行榜，证明你的预测能力
- 连续答对获得庆祝动画和成就

## 三种游戏模式

| 模式 | 说明 |
|------|------|
| 练习模式 | 无限练习，不计分 |
| 每日挑战 | 每天 5 题，与全国玩家比分数 |
| 挑战模式 | 计时挑战，测试真实实力 |

## 功能特点

- 📊 **真实历史事件** - 基于历史新闻与股价数据
- 🏆 **排行榜系统** - 每日挑战与挑战模式排行榜
- 🎯 **成就系统** - 解锁各种里程碑成就
- 📝 **错题本** - 回顾答错的题目
- 🌙 **暗色模式** - 保护眼睛
- 📱 **移动端优化** - PWA 支持，可添加到主屏幕
- 🔊 **音效反馈** - 答对/答错有音效提示

## 项目结构

```
trade_sense/
└── web/                 # 前端应用
    ├── src/
    │   ├── components/  # React 组件
    │   ├── hooks/        # 自定义 Hooks
    │   ├── models/       # 数据模型
    │   ├── services/     # 服务层
    │   └── utils/        # 工具函数
    └── public/           # 静态资源
```

## 本地运行

```bash
cd web
npm install
cp .env.example .env
# 填写必要的环境变量
npm run dev
```

## 技术栈

- **前端**: React 19 + TypeScript + Vite
- **样式**: Tailwind CSS + Framer Motion
- **状态**: React Hooks (useState, useEffect, useContext)
- **存储**: localStorage (匿名数据) + 可选 Supabase (用户数据)
- **部署**: Vercel

## CI/CD

- GitHub Actions 自动构建部署
- ESLint 代码检查
- Vitest 单元测试
